# MEMCREATOR 


Memcreator - это приложение по созданию картинок с приколом (мемов)
с системой уровней, написанная на Python 3.10.
____________________________________________________________________

Запуск:

Перед запуском распакуйте папку из .zip архива. Установите все нужные библиотеки
прописав команду "pip install -r requirements.txt" в консоли (терминале).
____________________________________________________________________

Функционал:

В наше время мемы - неотъемлемая часть жизни человека в сети, поэтому
memcreator облегчит создание смешных картинок для мемкреаторов. Но чтобы
спрос был больше реализована система уровней. С каждым новым сохраненным мемом
уровень повышается. Чем выше уровень, тем больше возможностей (плюшек) открывается,
например, увеличивается выбор шрифтов. 

Свой уровень можно посмотреть в меню пользователя (кнопка "Меню").
Там же отображается ваш логин, аватарка и имя. Здесь вы можете изменить свой пароль и аватарку.  

В приложении вы можете вставить в картинку текст, выбрав шрифт, размер и угол
наклона текста. Помимо текста есть возможность вставить картинку (пикчу) в мем
с выбранным размером, уровнем прозрачности и уголом наклона. Также есть возможность
вставить мем в "мем рамку", т.е. создание демотиватора.

Если у вас нет шаблона для мема, вы можете воспользоваться нашей библиотекой шаблонов.
